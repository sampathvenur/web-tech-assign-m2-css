CSS Relative and Absolute Positioning Example

Relative Positioning

This div is positioned relatively, which means it maintains its normal flow position but can be shifted using offsets.

Relative positioning allows other elements to flow around it.

Absolute Positioning

This div is positioned absolutely within its relatively positioned parent.

Absolute Box

Absolute positioning removes the element from the normal flow of the document and positions it relative to its nearest positioned ancestor (in this case, the parent with relative positioning).
