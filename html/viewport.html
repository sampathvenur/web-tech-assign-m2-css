Viewport Meta Tag

The viewport meta tag allows you to control the width and scaling of the viewport in mobile browsers.

Try resizing your browser window or viewing this page on a mobile device to see the effect.

The following viewport meta tag is used in this example:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

This meta tag sets the width of the viewport to the device width and sets the initial zoom level to 1.0.
